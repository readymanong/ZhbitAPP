
## 系统Android端的开发
Hello ZhbitAPP


## 主要内容

这是是教学评价与建议系统基于基于MVC设计的Android端的一款APP， 包含四个主要页面：

1. 知乎资讯页面：调用和解析知乎日报的ＡＰＩ；
2. 学院页面：采用Jsoup框架爬取学院公告栏的信息；
3. 主页页面：以Bmob作为系统后端，实现教师、学生和管理员的功能；
4. 我的页面：实现用户注册和登录功能。
